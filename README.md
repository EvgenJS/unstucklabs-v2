# UnstuckLabs v2

Monorepo for the UnstuckLabs suite: a public Store, an internal Admin panel, a shared core-api backend, and a growing set of independently-sold mini-app PWAs, each on its own subdomain. This is a from-scratch rewrite; v1 is archived at [github.com/EvgenJS/unstucklabs](https://github.com/EvgenJS/unstucklabs) for reference only.

See [docs/ROADMAP.md](docs/ROADMAP.md) for current phase and scope, and [CLAUDE.md](CLAUDE.md) for architecture and conventions.
